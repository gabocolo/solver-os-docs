# Aceleradores y agentizacion

Los **aceleradores** son bases reutilizables que se pueden adaptar a multiples dominios, industrias y clientes. Van mas alla de simples plantillas: son componentes estandarizados que aceleran la entrega.

La **agentizacion** es el proceso de identificar tareas repetitivas y automatizarlas mediante agentes de IA.

---

## Vision (Juan Gabriel Colorado)

> "No es esperar a que el cliente me de una necesidad. Yo miro mis procesos, miro los del cliente, y miro donde puedo participar y dar una buena idea."

### Principios
- Ser **proactivo**: identificar oportunidades antes de que el cliente las pida
- Crear agentes **transversales**: que sirvan a multiples equipos y empresas
- Quedar **ligado a la maduracion**: si implementas un agente transversal, muchos equipos lo usan
- La base es **reutilizable**, la personalizacion es **por cliente**

---

## Como se relacionan con el framework

Los aceleradores y agentes operan dentro del mismo marco:
- Usan **especificaciones** para definir su alcance
- Siguen la **gobernanza** del framework
- Se miden con las mismas **metricas**
- Pueden componer multiples **skills** en flujos automatizados
